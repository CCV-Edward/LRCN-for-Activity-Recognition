A implemetation of the LRCN in mxnet
=====

##Abstract
LRCN is a combination of CNN and RNN

##Installation
1. Download UCF101 dataset
2. mkdir vgg_model
3. Download the pretrained model of vgg16
4. python train_ucf101.py
